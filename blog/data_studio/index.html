Create COVID-19 Dashboard by Data studio

By Bruce Wu 10 March 2021

動機

疫情持續了一段時間，網上能看見各式儀表板，今天就試著來自己創建一個吧。

方法

透過將Python爬蟲程式部署至GCP，定期爬取疫情相關資料，再由Data studio創建儀表板直接讀取GCP中的DB 即可製作即時資訊儀表板
